Java 枚举

  • 枚举的定义

  • 枚举类型的静态方法valueOf很常用，它针对枚举类型而合成。例如

将Size.SMALL赋给notMySize。如果给定名称的实例不存在，valueOf会出IllegalArgumentException异常

  • 每个枚举类型都有一个静态方法values，该方法返回按照其声明次序排序的包含所有枚举实例的数组

这个方法也能用于foreach遍历

  • 可以给枚举类型添加构造函数、方法和域。每个枚举类型实例保证只被构造一次

  • 可以给单个实例添加方法，但是只能覆盖枚举类中定义的方法

  • 从技术上说，每个枚举常量都是枚举类的一个匿名子类，匿名子类体中的东西当然可以添加实现

  • 枚举可以有静态成员，但需要使用静态初始化块来进行初始化工作，因为枚举常量在静态成员之前构建

  • 枚举类型可以被嵌套在一个类中，这样嵌套的枚举类属于隐式的静态嵌套类，所以一样不能引用外部类的实例变量

  • 枚举常量可以被switch

java.lang.Enum类的方法

方法 描述
String toString() 在enum声明中所提供的该实例的名称
String name() 在enum声明中所提供的该实例的名称，name方法是final的
int ordinal() 在enum声明中该实例的位置
int compareTo(Emnu type,String name) 通过序数值与other实例进行比较
static T valueOf(Class type,String name) 返回给定名称的实例。考虑用枚举类型的valueOf或values方法替代它
Class getDeclaringClass() 获取实例所在的类
int hashCode() 调用对应的Object的hashCode方法并且是final的
protected void finalize() 调用对应的Object的finalize方法并且是final的
protected Object clone() 抛出一个cloneNotSupportedException异常
