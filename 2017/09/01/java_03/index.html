Java 内存分配

Java对内存空间的划分为五部分:
栈,堆,方法区,本地方法区,寄存器.

栈

栈内存中存储的是局部变量,一旦变量生命周期结束,该变量就被释放.

并不是数据结构中的栈.

堆

堆内存中存储的是实体(对象),每一个实体都有一个首地址值.new 会在堆内存中开辟空间.

堆内存中的变量都有一个默认初始化值.

数据类型 初始值
int 0
double 0.0
boolen false
char ‘\u0000’

堆中的空间释放依赖垃圾回收机制,具体查看多线程.
