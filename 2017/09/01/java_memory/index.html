Java 内存分配

JVM对内存空间的划分为五部分:
栈,堆,方法区,本地方法区,寄存器.

栈

  • 栈内存中存储的是局部变量,一旦变量生命周期结束,该变量就被释放.

  • 每个线程包含一个栈区, 栈中只保存基础数据类型的对象和自定义对象的引用

  • 栈区分为三个部分:基本类型变量区, 执行环境上下文, 操作指令区

  • 并不是数据结构中的栈.

堆(heap)

  • 堆内存中存储的是实体(对象),每一个实体都有一个首地址值.

  • 堆中不存放基本类型和对象引用

  • new 会在堆内存中开辟空间

  • JVM只有一个堆区并且所有线程共享

  • 堆内存中的变量都有一个默认初始化值.

数据类型 初始值
int 0
double 0.0
boolen false
char ‘\u0000’
  • 堆中的空间释放依赖垃圾回收机制,具体查看多线程.

方法区

  • 又称静态区, 跟堆一样, 被所有线程共享.

  • 这个方法包含所有的class和static变量

本地方法区

  • 运行时本地系统平台中的内容, 不同操作系统中也不一样

寄存器

  • 寄存器为计算机组成原理的内容.
