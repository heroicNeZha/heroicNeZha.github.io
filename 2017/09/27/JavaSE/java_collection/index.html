set

  • set可以有效检测一个值是不是它的元素，但是它不记得添加的顺序。
  • 例如你想知道一段文字是否含有关键词，可以这样实现。
  • 如果你为元素提供了很好的哈希函数，那么哈希set更加高效。Java类库中的String、Path等类都有很好的hash函数
  • TreeSet的元素必须实现Comparable接口，或者你在构造函数中提供Comparator

  • BitSet（位组）类可以用来存储一系列比特。
    • 位组将比特塞进一个long类型的数组里。所以存储值为boolean型的数组相比，使用位组更有效。
    • 第i个位置为1表示i在集合内。
    • 注意BitSet不是集合类，他没有实现Collection接口。
  • 栈、队列、双端队列、优先级队列
    • Java集合类框架中没有stack，只有一个Java早期遗留下来的应该避免使用的Stack类，如果你需要使用栈、队列和双端队列，并且不关心线程安全问题，可以使用ArrayDeque。
    • 使用ArrayDeque作为栈时，使用其push和pop方法，使用队列时使用add和remove方法。在并发程序中通常使用线程安全的队列。
    • 优先级队列在元素以任意顺序插入后，以有序方式检索元素。既当你调用remove方法时，会得到优先级队列中的当前最小元素。
