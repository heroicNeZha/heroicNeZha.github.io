Java 字符串

字符串就是一系列字符。

字符串连接

  • 字符串连接用＋
  • 以分隔符隔开String.join(分隔符,字符1,字符2)
  • 最好使用StringBuilder

子字符串

  • 子字符串拆开用substring(要提取的自字符串起始位置,不要的字符串起始位置)方法
  • 提取所有分割后的自字符串用split(分隔符)方法，分隔符可以是正则表达式

字符串比较

  • 不要使用==比较字符串，这种比较只有两个字符串在内存中为同一对象时才返回true。
    java虚拟机中每个文字串只有一个实例，因此"world"=="world"为真，但如果s1=s2.subString(7,12);则s1=="world"为假。

  • 测试是否为null可以使用==，但null和空字符串””不相同
  • 比较if("world".equals(s1))时，将文字串放前面更好，因为如果s1为null也可以正常工作
  • 不考虑大小写比较字符串使用equalsIgnoreCase方法
  • compareTo方法是比较Unicode的值

数字与字符串转换

  • String s = Integer.tostring(n,2)是将n的二进制数保存在字符串内
  • Integer.parseInt(str,2)将保存二进制数的字符串转换为int型
