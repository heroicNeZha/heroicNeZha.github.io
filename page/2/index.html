Java 数组

一维数组 定义方式:

  • 元素类型[] 数组名 = new 元素类型[元素类型或数组长度];

  • 声明变量是在栈内存中开辟空间 –> new 在堆内存中开辟空间 –> 赋值是将堆内存中
    实体的首地址赋给栈内存中的变量.所以数组是引用类型.

  • 数组编译时不会在堆分配空间,所以检查不出下标越界错误.

注意, 长度非零的数组总是可变的,所以类具有公共的静态final数组域, 或者返回这种域的访问方法,这几乎总是可变的.

多维数组 定义方式:

  • java没有真正意义上的多维数组，从技术上来讲，本质上二维数组是一维数组的数组

  • 元素类型[][] 数组名 = new 元素类型[一维长度][二维长度];

  • 在堆内存中一维实体存二维实体的首地址.

  • 如果二维数组长度不定,需要分别初始化二维数组.

数组算法

  • Arrays类和Collections类实现了数组和数组列表的常用算法
  • 填充Arrays.fill()
  • 排序Arrays.sort()，数组可以使用parallelSort方法，如果数组较大，该方法可以将工作分布到多个处理器上运行
  • 产生字符串Arrays.toString()
  • 二分查找Array.binarySearch(),查询成功返回正索引,查询失败返回-(1+插入位置);

数组列表

  • java.util.ArrayList类，其对象可以内部管理数组，当数组太小不够使用时，自动创建另外一个内部数组，并将元素移动到该数组，这个过程是透明的
  • 限制：不能使用基本类型作为类型参数，解决办法是使用包装类
  • 基本类型和它们对应的包装类型之间的转换是在一种名为自动装箱的过程中自动构造的
  • 将Integer对象赋值给int变量时，这个对象自动拆箱，内部产生一个int值

数组与数组列表的复制

  • 可以将数组变量复制到另一个数组变量，但两个变量引用同一个数组
  • 复制数组使用静态方法Arrays.copyOf()
  • 复制数组列表可以直接使用构造方法
  • Arrays.asList()方法可以使用一个数组或任意数量的参数，使用任意数量参数时，你可以将它当作初始化数组的一种替代，返回数组列表
  • 没有简单的方法可以完成int[]和ArrayList之间的转换，需要使用显示循环或IntStream

Java 基本数据类型

Java 中基本数据类型分整型,浮点型,字符型,布尔型四类八种 ;

整型
数据类型 长度 表示范围
byte 8 -128 ~ 127
short 16 -2^15 ~ (2^15)-1
int 32 -2^31 ~ (2^31)-1
long 64 -2^63 ~ (2^63)-1
  • 常量 Integer.MIN_VALUE和 Integer.MAX_VALUE分别是最小和最大的整数, 其他类也有同样作用的常量.
浮点型
数据类型 长度
float 32
double 64
  • Double.POSITIVE_INFINITY代表∞, Double.NEGATIVE_INFINITY代表-∞, Double.NaN代表”非数值”.
  • 所有的”非数值”都被认为是彼此不同的, 比如不能这样判断if(x==Double.NaN), 但是可以用if(Double.isNaN(x)).
字符型
数据类型 长度
char 16
布尔型
数据类型 内容 长度
boolean true or false 1字节

知识点

  1. Java低字节向高字节转换具有自动提升性质,转换顺序为 byte->short->int->long->float->double,但高字节向低字节强制转换可能丢失精度.

  2. 默认浮点类型为double,float加后缀f或F.

  • 可以给数字加下划线, 比如1_000_000, Java编译器会直接删除他们.
  1. Maths类提供常见的数学函数.
