Android 申请权限

我在AndroidManifest.xml中声明了十一种权限,其中ACCESS_COARSE_LOCATION,ACCESS_FINE_LOCATION,READ_PHONE_STATE,WRITE_EXTERNAL_STORAGE这四个权限需要进行运行时权限处理.
但是ACCESS_COARSE_LOCATION和ACCESS_FINE_LOCATION都属于同一个权限组,因此两者只要申请其一就可以了.

为了一次性申请三个权限,我创建了一个集合收集了没有被授权的权限,最后将集合转换成数组,再调用ActivityCompat.requestPermissions()方法一次申请.

除此之外,onRequestPermissionsResult对权限申请的逻辑处理也跟单独申请不同,所以在这里放置了Foreach循环对每个申请到的权限进行判断,如果被拒绝调用finish()方法关闭当前程序.

Java 多线程

进程:应用程序在内存中分配的空间
线程:是进程中负责程序执行的执行单元,也成为执行路径.一个进程中至少有一个线程在负责该进程的运行.
多线程:如果一个进程中有一个或多个线程在运行,就是多线程程序

Java IO

Java API 中读取字节被称为输入流,字节可以来自文件,网络链接中的数组(与Stream无关);

windows本身就可以操作设备,各种语言只是使用系统平台上的这个资源,并对外提供了各种语言自己的操作功能,这些资源使用完了一定要释放;

IO是用于处理设备上的数据的技术;Java中所涉及的功能对象都在java.io包中.

File

  • 设备上最常见的存储形式就是文件File

  • 使用的分隔符仅适用于Windows系统,全适配可以用File.separator

  • 同理换行符在windows下是\r\n,在linux下是\n,可以使用line.separator实现全适配.

  • 过滤文件可以使用文件过滤器FileFilter,如下是lambda表达式实现显示java文件

  • 删除文件夹的时候需要先删除文件夹的内容

流按操作分为两种: 字符流和字节流; 按流向分可分为两种: 输入流和输出流
